# Instagram Bot With an AI API
> Chat with an AI on Instagram


## Installation

```sh
npm install @androz2091/insta.js
```

## Setup
> CREATE A ENV FILE WITH 

USERNAME , PASSWORD , AND API KEY,  
You can get the API key from [here](https://api.monkedev.com)

>START 
```sh
node main.js
```
